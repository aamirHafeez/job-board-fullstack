# Project 1: Full-Stack Job Board Web App

Build a production-quality full-stack Job Board web application that demonstrates frontend, backend, API, database, filtering, and CRUD skills. The app should be portfolio-ready, beginner-friendly to run, and fully free to build and deploy.

## Tech Stack

- Frontend: React + Vite
- Styling: Plain CSS
- Backend: Node.js + Express
- Database: MySQL
- API: REST API
- Version Control: Git
- Deployment: Vercel for frontend, Render or Railway for backend

## Core Features

- Home page with hero section and featured jobs
- Jobs listing page with search by title or keyword
- Filters for location, category, job type, and experience level
- Single job details page
- Employer dashboard to add, edit, and delete job posts
- Frontend and backend validation
- Responsive layout for mobile, tablet, and desktop
- Loading states, empty states, and friendly error messages
- Seed data so the app works immediately
